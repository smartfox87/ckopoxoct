@@include("../symbol/symbol.html", { "name": "servers--ssd" }) Выделенные SSD

Предоставляем для серверов выделенные SSD с гарантированной пропускной способностью, а не дисковые полки с плавающими параметрами, зависящими от загрузки “соседних” виртуальных серверов, которые также используют данную полку.

@@include("../symbol/symbol.html", { "name": "servers--power" }) Резервирование электропитания

Электропитание обеспечивается двумя независимыми вводами от разных электростанций и дизель-генераторной установкой.

@@include("../symbol/symbol.html", { "name": "servers--data" }) Сертифицированный дата-центр

Дата-центр прошел сертификацию по стандарту безопасности PCI-DSS.

@@include("../symbol/symbol.html", { "name": "servers--cluster" }) Кластеризация

Если основной сервер выйдет из строя, работа ресурса не прервется, благодаря запасному серверу в кластере. Серверы размещаются в разных дата-центрах, для обеспечения высокой доступности и отказоустойчивости.

@@include("../symbol/symbol.html", { "name": "servers--mirror" }) Зеркалирование

Raid-системы для дублирования данных на нескольких винчестерах. Более высокая скорость чтения и защита от сбоя железа.

@@include("../symbol/symbol.html", { "name": "servers--internet" }) Многоканальный интернет

Оптоволоконная инфраструктура построена по системе «кольцо». Подключение к каналам связи на скорости 100/1000 Мбит/с.

@@include("../symbol/symbol.html", { "name": "servers--security" }) Безопасность данных

Комплекс мер, включая: протоколирование доступа в дата-центр (физически) и на сервер (протоколы), копирование данных на случай утраты, специализированное ПО для проактивной защиты.

@@include("../symbol/symbol.html", { "name": "servers--protection" }) Сетевая защита и лучший файервол

Мониторинг трафика файрволом, блокировки/разблокировки портов по запросу, установка специализированного ПО по запросу.
